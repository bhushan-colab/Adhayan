HTML Introduction

HTML (HyperText Markup Language) is the standard language used for creating web pages. It forms the foundation of web development and is used to structure content on the web.

Basic Syntax of HTML

Here is a simple example of an HTML document structure:





    My First Web Page


    
Welcome to My Website

    
This is my first web page created with HTML.

Why Learn HTML?

  • HTML is the foundation of web development.
  • It is easy to learn and understand.
  • HTML allows you to create structured content for websites.

HTML Elements

Some essential HTML elements:

  • <html>: The root element that defines the HTML document.
  • <head>: Contains metadata and links to external resources.
  • <body>: Contains the visible content of the webpage.
  • <h1> to <h6>: Define headings with different levels.
  • <p>: Defines a paragraph of text.
  • <a>: Defines a hyperlink to other pages.
  • <ul> <ol>: Define unordered and ordered lists.
  • <img>: Embeds an image in the page.

Explanation of the Program:
