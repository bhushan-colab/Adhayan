CSS Introduction

CSS (Cascading Style Sheets) is a language used to describe the presentation of a web page, including its layout, colors, fonts, and spacing. It works alongside HTML, which provides the structure of the page, to make web content visually appealing. While HTML defines the structure of a webpage (headings, paragraphs, lists, etc.), CSS controls its style and layout. With CSS, you can change the appearance of the content, adjust how elements are positioned on the page, and apply various styles to create a beautiful, user-friendly design.

Basic Syntax of CSS


CSS consists of selectors and declarations. Hereâ€™s the general syntax: css selector { property: value; }
Selector: Targets the HTML element(s) you want to style. Property: Specifies what aspect of the element you want to change. Value: The value for that property. 
    
        
Why Learn Css ?

        1.Separation of Content and Design:CSS separates content (HTML) from design. This makes it easier to update and maintain the look of a website without touching its underlying structure.

        2.Control over Layout: With CSS, you can control the positioning of elements, and make responsive layouts that adapt to different screen sizes.

        3.Consistent Styling: You can apply styles consistently across multiple pages of a website, making it easier to maintain a uniform look and feel.

        4.Advanced Visual Effects: CSS allows you to add visual effects like transitions, animations, shadows, and gradients.

     CSS Elements
    
        CSS (Cascading Style Sheets) is a language used to describe the style and layout of web pages. CSS defines the visual appearance of HTML elements such as colors, fonts, spacing, positioning, and responsiveness. 
    Here's an overview of the core CSS elements you can use:
    
    1.Selectors:-
     
        
  • Universal Selector (*): Targets all elements in a document.
    • 
            * {
                margin: 0;
                padding: 0;
            }
        
  • Type Selector (Element Selector): Targets all elements of a specific type (e.g., all 
     elements).
    • 
              p {
                  color: #333;
                }
    
        
  •  Class Selector (.): Targets elements with a specific class attribute.
    • 
                .container {
                    width: 80%;
                    margin: 0 auto;
    
        
  • ID Selector (#):Targets an element with a specific ID attribute (IDs should be unique).
    • 
              #header{
                      background-color: #004A6C;
                color: white;
                     }
    
         
  • Attribute Selector: Targets elements based on an attribute or attribute value.
                input[type="text"] 
                {
                     border: 1px solid #ccc;
                }
    
         
  • Descendant Selector (space): Targets an element that is a descendant of another element.
                div p {
                    font-size: 16px;
                      }
    
         
  • Child Selector (>):Targets direct child elements of a parent.
    
                div > p {
                     margin-top: 20px;
                        }
     2. Properties:-
    
            CSS properties define the style of the selected elements. Here's a list of common properties you will use to style elements:
            
         
  • Text Properties:
    •          
            
  • color: Sets the text color.
    •  
                p {
                    color: #333;
                }
    
        
  • font-family: Sets the font of the text.
    • 
            h1 {
                font-family: 'Arial', sans-serif;
            }
            
        
  • font-size: Sets the size of the text.
    • 
            p {
                font-size: 16px;
            }
    
         
  •  Box Model Properties: 
            
  • width: Sets the width of an element.
    • 
    .box {
        width: 100px;
    }
    
    
  • height: Sets the height of an element.
    • 
    .box {
        height: 200px;
    }
    
    
  • padding: Sets space inside the element, between the content and the border.
    • 
    .box {
        padding: 20px;
    }
    
    
  • margin: Sets space outside the element, between the element's border and surrounding elements.
    • 
    .box {
        margin: 20px;
    }
    
    
  • border: Sets the border around the element.
    • 
    .box {
        border: 2px solid #000;
    }
    
    
  • Background Properties:
    • 
        
  • background-color: Sets the background color of an element.
    • 
    div {
        background-color: #f4f4f4;
    }
    
    
  • background-image: Sets an image as the background.
    • 
    div {
        background-image: url('background.jpg');
    }
    
    
  • background-position: Defines the position of a background image.
    • 
    div {
        background-position: center;
    }
    
    
  • Layout and Positioning:
    • 
    
  • display: Defines how an element is displayed.
    • 
    div {
        display: block; /* or inline, flex, grid */
    }
    
    
  • position: Sets the positioning method of an element (static, relative, absolute, fixed).
    • 
    .box {
        position: absolute;
        top: 50px;
        left: 20px;
    }
    
    
  • top, right, bottom, left: Set the distance of an element from its containing element when position is not static.
    • 
    .box {
        position: relative;
        top: 10px;
        left: 20px;
    }
    
    
  • float: Makes an element float to the left or right.
    • 
    .image {
        float: left;
        margin-right: 20px;
    }
    
    
  • Flexbox and Grid Layout:
    • 
        
  • display: flex: Activates a flex container.
    • 
    .container {
        display: flex;
        justify-content: space-between;
    }
    
    
  • flex-direction: Defines the direction of the flex items.
    • 
    .container {
        flex-direction: row; /* or column */
    }
    
    
  • grid-template-columns: Defines the number and size of columns in a grid container.
    • 
    .grid {
        display: grid;
        grid-template-columns: repeat(3, 1fr);
    }
    
     3. Pseudo-Classes
    
    Pseudo-classes are used to define the special states of an element:
    
  • :hover: Applied when an element is hovered over.
    • 
    a:hover {
        color: red;
    }
    
    
  • :active: Applied when an element is activated (e.g., clicked).
    • 
    button:active {
        background-color: #ccc;
    }
    
    
  • :focus: Applied when an element (e.g., form input) gains focus.
    • 
    input:focus {
        border-color: blue;
    }
    
    
  • :nth-child(): Targets specific children within a parent element.
    • 
    li:nth-child(2) {
        color: blue;
    }
    
    
  • :first-child / :last-child: Targets the first or last child element.
    • 
    p:first-child {
        font-weight: bold;
    }
    
    4. Pseudo-Elements
    Pseudo-elements allow you to style specific parts of an element:
    
  • ::before: Inserts content before an element's actual content.
    • 
    p::before {
        content: "Note: ";
        font-weight: bold;
    }
    
    
  • ::after: Inserts content after an element's actual content.
    • 
    p::after {
        content: ".";
    }
    
    5. Media Queries
    Media queries allow you to apply CSS styles based on the device's characteristics (e.g., screen size).
    @media (max-width: 768px) {
        body {
            background-color: lightblue;
        }
    }
