Python basic syntax

    Variables:-Created by assigning values directly.
    Comments:-Use # for single-line comments and triple quotes for multi-line comments.
    Indentation:- Essential for defining blocks of code.
    Control Flow:- Uses if, for, and while for conditional and looping statements.
    Functions:-Defined using def, can return values with return.
    Data Structures:- Includes lists, tuples, dictionaries, and sets.
    Modules:-Can be imported using import and from ... import.
    Error Handling:- Uses try, except, and finally.
    OOP:-Supports classes, objects, and inheritance.
    Lambda:- Use lambda for small anonymous functions.
    Input/Output:- input() for user input, print() for output.

python Example: A Simple hello world program

     # Function to calculate factorial
        def factorial(n):
            if n == 0 or n == 1:
            return 1
            else:
            return n * factorial(n - 1)
        
        # Get input from user
        num = int(input("Enter a number to find its factorial: "))
        
        # Calculate and display the result
        if num < 0:
            print("Factorial does not exist for negative numbers.")
        else:
            result = factorial(num)
            print(f"The factorial of {num} is {result}.")

Explanation of the Program:

For n = 5, the function computes:
factorial(5) = 5 * factorial(4)
factorial(4) = 4 * factorial(3)
factorial(3) = 3 * factorial(2)
factorial(2) = 2 * factorial(1)
factorial(1) = 1 (Base case)
Thus, factorial(5) = 5 * 4 * 3 * 2 * 1 = 120.

How to Compile and Run:

1.Install Python if not already installed.
2.Write your Python program in a .py file.
3.Open the terminal/command prompt and navigate to the folder containing the .py file.
4.Run the program using the command python3 my_program.py (or python my_program.py on some systems).
5.See the output in the terminal/command prompt.

Output:
