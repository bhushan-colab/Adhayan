JavaScript Introduction

JavaScript is a versatile programming language primarily used to add interactive behavior to websites. Developed by Brendan Eich in 1995, JavaScript has become a core technology of web development along with HTML and CSS.

Basic Syntax of JavaScript

Below is a simple JavaScript program that displays a message in the browser console.


            // JavaScript Program
            console.log("Hello, World!");

Why Learn JavaScript?

JavaScript is one of the most widely-used programming languages for web development, enabling dynamic behavior on websites.

If you know JavaScript, you can work on both front-end and back-end development (with Node.js).

JavaScript Elements

Summary of Key JavaScript Elements:

  • Variables and Data Types: Declare variables (let, const, var) and use data types (number, string, boolean, object).
  • Operators: Perform arithmetic, logical, and comparison operations.
  • Control Flow: Use if, else, switch, loops (for, while, do-while) to control the program flow.
  • Functions: Modularize your code by creating reusable blocks.
  • Objects: Use objects to store related data and functionality.
  • Arrays: Use arrays to store multiple values of the same type.
  • DOM Manipulation: Manipulate HTML elements dynamically using JavaScript.
  • Event Handling: Handle events such as clicks, keypresses, and mouse movements.
  • Asynchronous Programming: Handle asynchronous code with promises and async/await.

JavaScript Example: A Simple Hello World Program


            // JavaScript Program to Print "Hello, World!"
            console.log("Hello, World!");

Explanation of the Program:

console.log("Hello, World!");: This line of code prints the message "Hello, World!" to the browser's console.

How to Run JavaScript:

1. Save the program in an HTML file, for example, hello.html.

2. Open the file in any browser.

3. Open the browser's developer tools (usually F12), and go to the Console tab to see the output.

Output:

Hello, World!
